# Restaurant Management System (Python)

This is a console-based Restaurant Management System developed using Python.

## Features
- Show menu
- Add food items to order
- Update ordered items
- Remove items from order
- Search ordered items
- Calculate total bill

## Technologies Used
- Python
- Object-Oriented Programming (OOP)

## How to Run
1. Clone the repository
2. Run `restaurant.py`
3. Follow the menu options in the console
